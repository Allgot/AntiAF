Setup smartphone and computer as explained in `README_android_scripts.md`.
1. Requirements
  * Instagram app must be installed and instagram account already logged in.
  * You must have some followed accounts (otherwise home will be empty)
  * In case you are using a new and empty account you can follow accounts in `top_500_instagram.txt`.
2. Edit `instagram_push_like.py` (if needed), parmeters:
  * `likes`, how many likes before stopping (default: 100)
3. Run script from terminal: `python instagram_push_like.py`
